Karma Online Shoes Store

A .NET project about an online shoes e-commerce shop done in group of 3 students.

Allows users to view shoes based on related information, order shoes, view order history, as well as managing shoes.

  • 3-tier architecture design using Amazon EC2 web service, Amazon remote database and MVC .NET Core 2.1
  • Friendly interfaces for managing products with DataTable
  • Faster response time using AJAX and JSON
